# Technical Research Assistant

The [agent_config.yaml](./agent_config.yaml) and the [.mcp.json](./.mcp.json) files available in this folder can be used to create a technical research assistant leveraging [DeepWiki MCP](https://docs.devin.ai/work-with-devin/deepwiki-mcp).

Example prompt:

```md
Could you write a report on the Agent Client Protocol (or ACP)? GitHub Repo: https://github.com/agentclientprotocol/agent-client-protocol
```
